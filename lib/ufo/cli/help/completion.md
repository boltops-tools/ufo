Example:

  ufo completion

Prints words for TAB auto-completion.

## Examples

  ufo completion
  ufo completion ship
  ufo completion docker

To enable, TAB auto-completion add the following to your profile:

  eval $(ufo completion_script)

Auto-completion example usage:

  ufo [TAB]
  ufo ship [TAB]
  ufo docker build [TAB]
  ufo docker build --[TAB]
